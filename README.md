# Annie's SQL Adventure 🐕

A full beginner SQL course built around **Annie** and her dog park — 22 lessons, achievements, confetti, and real photos.

## Quick start

```bash
npm install
npm run dev
```

Open `http://localhost:5173` and hard-refresh if needed (`Ctrl+Shift+R`).

## Course outline (~4–5 hours)

| Part | Topics | Lessons |
|------|--------|---------|
| 1 | SELECT, WHERE, ORDER BY, COUNT | 6 |
| 2 | AND/OR, LIKE, IN, DISTINCT | 4 |
| 3 | SUM, AVG, GROUP BY, HAVING | 3 |
| 4 | INNER JOIN, LEFT JOIN, 3-table join | 3 |
| 5 | Aliases, LIMIT, CASE, subqueries | 4 |
| 6 | Midterm & final challenges | 2 |

## Features

- Interactive SQL playground (runs in the browser)
- Progress bar and time estimates
- **13 achievements** with confetti celebrations
- Progress saved in localStorage

## Customize

- Lessons: `src/lessons/`
- Achievements: `src/achievements.ts`
- Data: `src/db/initDb.ts`
- Photos: `public/annie/`

## Build

```bash
npm run build
npm run preview
```
